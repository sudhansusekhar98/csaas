# Quickstart: Child Sample Count, Operator Assignment & Lab Receive Confirmation

## Scenario 1 — Division Station: Custom Child Sample Count

1. Open Division Station → open session `DIV-8822-X` (IN_PROGRESS)
2. Complete Pulverisation Log (mark as opened)
3. Small Bagging section appears — a "Number of Child Bags" input shows (default 3)
4. Change the count to 4 — a 4th bag card "Sample D" appears
5. Enter weights and assign operators to all 4 bags
6. Click "Seal All Child Bags" — all 4 bags show sealed status with QR seal IDs and assigned operators

## Scenario 2 — Division Station: Operator Assignment Required

1. In Small Bagging, fill in weights but leave one bag's operator unselected
2. "Seal All Child Bags" remains disabled
3. Select all operators → button enables → click to seal

## Scenario 3 — Lab Receiving: Confirm Receipt

1. Open Lab Receiving
2. In Batch Verification panel, fill in: Lab Receiver (S. Reddy), Acceptance Status (Accepted), Visual Condition (Intact)
3. "Confirm Receipt" button activates
4. Click it — the first pending item in the table changes to "Received" status, form resets, success message shows briefly
